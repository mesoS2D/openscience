Example filenames:
nonrr_cm_mean_T_NO_O_CO2_CO_H2O_fullcol_2010-06-01-00000.nc
rr_cm_mean_T_NO_O_CO2_CO_H2O_fullcol_2010-06-01-00000.nc

File naming format is:
{RR/Non-RR}_{area}_mean_{list of variables in file}_{"fullcol" if all levels from surface to TOA are included}_{yyy-mm-dd}-{#####}.nc
